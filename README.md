# Deep Learning Image Classification

## Project Overview

This project uses a Deep Learning Neural Network to classify fashion images from the Fashion MNIST dataset.

## Dataset

Fashion MNIST Dataset

Training Images: 60000

Testing Images: 10000

## Technologies Used

* Python
* TensorFlow
* Keras
* Matplotlib

## Model Architecture

* Flatten Layer
* Dense Layer (128 Neurons, ReLU)
* Dense Output Layer (10 Classes, Softmax)

## Training

* Optimizer: Adam
* Loss Function: Sparse Categorical Crossentropy
* Epochs: 10

## Outputs

* Model Accuracy Graph
* Test Accuracy
* Saved Model File

## Files Included

* Task2\_Deep\_Learning\_Image\_Classification.ipynb
* Task2\_Report.docx
* Task2\_Screenshots
* fashion\_mnist\_model.keras

## Conclusion

The Deep Learning model successfully classified fashion images and achieved good performance on the test dataset.
